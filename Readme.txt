# Hao Shi Guang - Restaurant Website

A modern restaurant website built with **Next.js**, showcasing the menu, seasonal dishes, and allowing customers to explore more about the restaurant.  
The project includes a "Join Our Team" page with an application form that sends emails (with resume attachments) using **Nodemailer**.

---

## **Features**
- **Responsive Design** – Works on mobile, tablet, and desktop.
- **Dynamic Menu** – Displays featured dishes and full menu items.
- **Seasonal Dishes** – Special picks with images and prices.
- **Contact & Application Form** – Sends email applications with attachments.
- **Optimized for Performance** – Using Next.js static generation and server-side rendering.
- **Ready for Deployment** – Configured for Vercel hosting.

---

## **Tech Stack**
- **Frontend:** Next.js, React, Swiper.js (sliders), Tailwind/Custom CSS
- **Backend:** Next.js API routes (Node.js)
- **Email:** Nodemailer with Gmail or SendGrid
- **Hosting:** Vercel
